# Membership & Map App

## 2016/02/29
## Synopsis

A simple app including:
* login screen
* user signup
* user manager 
* map screen plotting addresses of all users.

## View it live
https://whispering-citadel-54738.herokuapp.com/login

## Motivation
This was an exercise in better organizing my MVC code in Angular.
